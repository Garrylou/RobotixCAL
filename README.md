# RobotixCAL

Extension MakeCode pour micro:bit qui regroupe les blocs **Nezha 2** et **PlanetX**
d'ELECFREAKS, avec les noms de blocs traduits en français.

Conçue pour l'enseignement de la robotique au secondaire.

## Contenu

- **Nezha 2** : moteurs et servos (catégorie « Nezha 2 » dans la palette).
- **PlanetX** : capteurs et modules de la série PlanetX (catégorie « PlanetX »).

Les deux extensions s'installent en une seule étape : il suffit d'importer
RobotixCAL et les deux apparaissent automatiquement dans la palette de blocs.

## Installation dans MakeCode

1. Ouvrir un projet sur https://makecode.microbit.org
2. **Avancé → Extensions**
3. Coller l'URL : `https://github.com/garrylou/RobotixCAL`
4. (Optionnel) Régler la langue de l'éditeur en français (icône engrenage → Language → Français)
